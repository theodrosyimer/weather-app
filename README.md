# Weather App

## Context

Formation Assofac DWWM 2022-2023
> Project : create a mobile only weather app.

## Usage

**Install**

```sh
git clone https://github.com/theodrosyimer/weather-app.git
```

**Dev**

```sh
npm run dev
```

**Build**

```sh
npm run build
```

**Preview**

if the project is already built:

  ```sh
  npm run preview
```

otherwise do both:

```sh
  npm run build && npm run preview
```
